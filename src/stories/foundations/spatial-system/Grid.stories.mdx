# Grid System

## Usage

Langkah awal yang harus dilakukan sebelum menggunakan class utilities dari grid system adalah dengan meng-import gamatechno component library style file ke dalam pada file `main.js` atau root file pada projek.

```js
import "gamatechno-vue-component-library/css/foundations/style.css";
```

Untuk menggunakan utility class dari grid system pada sebuah element atau komponen dapat dengan menuliskan nama classnya seperti contoh di bawah ini:

<img src={GridExampleImg} alt="grid example" />

```html
<div class="grid--container text-center">
  <div class="grid--row">
    <div class="grid--col-lg-12 bg--teal-300">1</div>
  </div>
  <div class="grid--row">
    <div class="grid--col-sm-2 bg--teal-300">1</div>
    <div class="grid--col-sm-2 bg--yellow-300">2</div>
  </div>
  <div class="grid--row">
    <div class="grid--col-lg-4--gu-sm bg--teal-300">1</div>
    <div class="grid--col-lg-4--gu-sm bg--yellow-300">2</div>
    <div class="grid--col-lg-4--gu-sm bg--green-300">3</div>
  </div>
  <div class="grid--row">
    <div class="grid--col-md-2--gu-md bg--teal-300">1</div>
    <div class="grid--col-md-2--gu-md bg--yellow-300">2</div>
    <div class="grid--col-md-2--gu-md bg--green-300">3</div>
    <div class="grid--col-md-2--gu-md bg--red-300">4</div>
  </div>
  <div class="grid--row">
    <div class="grid--col-lg-2--gu-lg bg--teal-300">1</div>
    <div class="grid--col-lg-2--gu-lg bg--yellow-300">2</div>
    <div class="grid--col-lg-2--gu-lg bg--green-300">3</div>
    <div class="grid--col-lg-2--gu-lg bg--red-300">4</div>
    <div class="grid--col-lg-2--gu-lg bg--mustard-300">5</div>
    <div class="grid--col-lg-2--gu-lg bg--navy-blue-300">6</div>
  </div>
</div>
```

## List of Utility Classes

### Grid Column Type List

<br />

#### Small Grid

Grid ini digunakan pada perangkat dengan ukuran layar kurang dari `480px` (small devices)

| Class          | Description    |
| -------------- | -------------- |
| grid--col-sm-1 | 1 column width |
| grid--col-sm-2 | 2 column width |
| grid--col-sm-3 | 3 column width |
| grid--col-sm-4 | 4 column width |
