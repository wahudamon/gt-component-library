# Gamatechno Component Library

Gamatechno Component Library adalah sebuah tools berisi kumpulan komponen yang dapat digunakan oleh pengembang aplikasi dari PT Gamatechno Indonesia dalam menyusun layout produk digital yang sedang dikembangkan pada platform web dengan lebih mudah dan cepat.

## Installation

Agar dapat menggunakan library ini di produk atau projek yang sedang dibangun, library ini harus diinstal terlebih dahulu menggunakan beberapa cara dibawah ini:

### Install menggunakan NPM

```bash
  npm install gamatechno-vue-component-library
```

### Install menggunakan Yarn

```bash
  yarn add gamatechno-vue-component-library
```
