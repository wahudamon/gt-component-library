# Margin

## Usage

Langkah awal yang harus dilakukan sebelum menggunakan class utilities dari margin adalah dengan meng-import margin styles list dari package ke dalam section style komponen target.

```html
<style lang="scss">
  @import "gamatechno-vue-component-library/src/styles/margin";
</style>
```

Untuk menggunakan utility class dari margin pada sebuah element atau komponen dapat dengan menuliskan nama classnya seperti contoh di bawah ini:

```html
<div class="m--1"></div>
<div class="mx--2"></div>
<div class="my--3"></div>
<div class="mt--4"></div>
<div class="mb--5"></div>
<div class="ml--6"></div>
<div class="mr--7"></div>
```

## List of Utility Classes

### Margin Type

| Class       | Description                     |
| ----------- | ------------------------------- |
| m--(value)  | margin top, right, bottom, left |
| mx--(value) | margin horizontal (left, right) |
| my--(value) | margin vertical (top, bottom)   |
| mt--(value) | margin top                      |
| mb--(value) | margin bottom                   |
| ml--(value) | margin left                     |
| mr--(value) | margin right                    |
