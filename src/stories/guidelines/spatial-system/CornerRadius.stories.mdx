# Corner Radius

Untuk desainer dan developer selalu gunakan nilai radius sudut yang telah ditentukan sebelumnya untuk membulatkan elemen umum seperti cards, buttons dan messages. Nilai alternatif harus dihindari kecuali saat membuat bentuk elemen yang custom.

Kemudian perlu dingat bahwa tidak semua komponen memerlukan radius sudut. Dan radius sudut yang digunakan oleh **GT Design Team Library** akan mengikuti sistem gird yang digunakan yaitu menggunakan nilai kelipatan 8px.

## Usage

Langkah awal yang harus dilakukan sebelum menggunakan class utilities dari corner radius styles list adalah dengan meng-import corner radius dari package ke dalam section style komponen target.

```html
<style lang="scss">
  @import "gamatechno-vue-component-library/src/styles/corner-radius";
</style>
```

Untuk menggunakan utility class dari corner radius pada sebuah element atau komponen dapat dengan menuliskan nama classnya seperti contoh di bawah ini:

```html
<div class="cr--md"></div>
```
