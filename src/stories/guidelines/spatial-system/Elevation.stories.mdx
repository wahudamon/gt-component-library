# Elevation

Elevation membawa kedalaman pada aplikasi dan mewakili tingkat kepentingan secara berbeda sesuai dengan komponen tertentu. Ini lebih menekankan pada sebagian besar elemen antarmuka (UI) aplikasi.

## Usage

Langkah awal yang harus dilakukan sebelum menggunakan class utilities dari elevation adalah dengan meng-import elevation styles list dari package ke dalam style tag yang sudah ditambahkan `lang="scss"` pada file `App.vue`.

```css
@import "gamatechno-vue-component-library/css/elevation";
```

Untuk menggunakan utility class dari elevation pada sebuah element atau komponen dapat dengan menuliskan nama classnya seperti contoh di bawah ini:

```html
<div class="be--low"></div>
<div class="te--high"></div>
```
